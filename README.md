# Magic Circle Game

魔法陣グルグルにインスパイアされた魔法陣召喚バトルゲーム。
指で魔法陣（円）をなぞり、その正確さと速さが攻撃力を決める。

## 遊び方

1. キャラクターを選ぶ（炎・雷・氷）
2. 画面のガイド円に沿って指でなぞる
3. 正確さ × 速さ = ダメージ
4. 3ラウンドでダークドラゴンを倒せ！

### ランク
| ランク | 精度 |
|--------|------|
| PERFECT!! | 90%以上 |
| GREAT! | 75〜89% |
| GOOD | 55〜74% |
| MISS | 30〜54% |
| FAIL | 30%未満 |
| 💀 禁呪発動！ | FAIL + 確率発動 → 大ダメージ |

## キャラクター

| キャラ | 属性 | 特性 |
|--------|------|------|
| 🔥 アギ | 炎 | 速さボーナス最大3倍 |
| ⚡ ライ | 雷 | 高精度でダメージ1.2倍 |
| ❄️ フィオ | 氷 | 禁呪発動率50% |
